Shared constraints
Dark theme, system font stack, 3-size type scale, tabular numerals for all values.
Colors limited to palette: white, muted grays, green-medium, red, yellow.
Cards use subtle borders + 1px inner highlight instead of heavy shadow. Status communicated via color + text + icon.
All mockups assume the indexer is the source of truth for tracking — history and in-flight status panels are indexer-driven.
